# Cloud Computing

## Definition

### English (Source 1 - Microsoft)

> Cloud Computing is the delivery of computing services—including servers, storage, databases, networking, software, and analytics—over the Internet.

### English (Source 2 - AWS)

> Cloud Computing provides on-demand access to IT resources through the Internet with pay-as-you-go pricing.

### Thai

Cloud Computing คือ เทคโนโลยีที่ให้บริการทรัพยากรด้านคอมพิวเตอร์ผ่านอินเทอร์เน็ต เช่น เซิร์ฟเวอร์ พื้นที่จัดเก็บข้อมูล ฐานข้อมูล เครือข่าย และซอฟต์แวร์ โดยผู้ใช้งานสามารถเข้าถึงบริการเหล่านี้ได้ทุกที่ทุกเวลาโดยไม่จำเป็นต้องติดตั้งหรือดูแลโครงสร้างพื้นฐานด้วยตนเอง

---

## Explanation

Cloud Computing เป็นรูปแบบการให้บริการทรัพยากรคอมพิวเตอร์ผ่านเครือข่ายอินเทอร์เน็ต ช่วยให้องค์กรและผู้ใช้งานสามารถเข้าถึงข้อมูลและแอปพลิเคชันได้จากทุกที่

### องค์ประกอบสำคัญ

1. **Infrastructure as a Service (IaaS)**

   * ให้บริการโครงสร้างพื้นฐาน เช่น เซิร์ฟเวอร์ ระบบเครือข่าย และพื้นที่จัดเก็บข้อมูล

2. **Platform as a Service (PaaS)**

   * ให้บริการแพลตฟอร์มสำหรับพัฒนาและทดสอบแอปพลิเคชัน

3. **Software as a Service (SaaS)**

   * ให้บริการซอฟต์แวร์สำเร็จรูปผ่านอินเทอร์เน็ต

4. **Cloud Storage**

   * บริการจัดเก็บข้อมูลบนระบบคลาวด์ที่สามารถเข้าถึงได้จากทุกอุปกรณ์

### ตัวอย่างการใช้งาน

* Google Drive
* Microsoft OneDrive
* Dropbox
* Microsoft 365
* Amazon Web Services (AWS)

### Keywords

`Cloud Computing` `Cloud Storage` `IaaS` `PaaS` `SaaS` `Virtualization`
